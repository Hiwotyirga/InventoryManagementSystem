# Inventory Management System (Ellatech Take-Home Exercise)

A modern, responsive inventory management application built with **React Native (Expo)** and **NativeWind**. This app demonstrates full-stack logic on the frontend, focusing on state management, audit trails, and clean UI/UX.

## 🚀 Features
- **User Authentication:** Simulated registration and profile management.
- **Product Management:** Full product registration including SKU, pricing, and initial stock.
- **Live Inventory Control:** Real-time stock adjustment with built-in validation (prevents negative inventory).
- **Transaction History:** Comprehensive audit log (Activity Log) tracking every registration and stock movement.
- **Product Status:** Detailed view for items showing SKU, current stock levels, and "Last Updated" timestamps.

## 🛠️ Tech Stack
- **Framework:** Expo (React Native)
- **Routing:** Expo Router (File-based routing with `(auth)` and `(tabs)` groups)
- **Styling:** NativeWind (Tailwind CSS for React Native)
- **State Management:** React Context API
- **Icons:** @expo/vector-icons (Ionicons)
